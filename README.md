# RetailOps Intelligence

RetailOps Intelligence is an AI-powered retail operations command center for inventory monitoring, sales analytics, demand forecasting, supplier scoring, and stockout-risk alerts.

## Problem
Retail teams struggle with:
- stockouts
- overstocking
- poor demand planning
- supplier delays
- slow sales visibility
- manual inventory decisions

## Solution
RetailOps Intelligence simulates a retail operations platform that tracks product inventory, analyzes sales trends, forecasts demand, detects low-stock risk, scores suppliers, and generates downloadable operations reports.

## Core Features
- Executive KPI dashboard
- Product inventory dashboard
- Sales trend analytics
- Low-stock alert system
- Demand forecasting
- Supplier performance tracking
- Data quality / pipeline monitoring
- Downloadable operations report
- Self-initializing Streamlit app

## Run Locally

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
streamlit run app/streamlit_app.py
```

## Tech Stack
Python, Streamlit, Pandas, NumPy, scikit-learn, Plotly.
